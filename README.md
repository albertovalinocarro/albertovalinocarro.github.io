# 💼 Personal CV Website — Alberto Valiño Carro

My interactive CV website, built with React, TypeScript, Tailwind, Vite, and deployed on GitHub Pages.
Backed by Vercel serverless functions and Supabase, with AI-powered translation, analytics, and an interactive terminal.

Live site: 👉 https://albertovalinocarro.github.io/

---

## ✨ Features

- 🌙 Dark/Light mode
- 📱 Responsive and mobile-friendly design
- 🌐 AI-powered translation (EN ↔ ES) with caching + versioning
- 📊 Visitor analytics — total visits + unique visitors (Supabase)
- 🚀 CI/CD badge (last commit via GitHub API)
- 🎨 Polished UI — animated skill badges, timeline cards, typewriter hero, tooltips
- 💻 Interactive terminal — type `help` to explore
- 🔍 Observability with LangSmith (traced AI calls)
- 🔒 Snyk integration for dependency and container security

---

## 🛠️ Tech Stack

- **Frontend**: React 19 + TypeScript + Tailwind v4 + Framer Motion
- **Backend**: Vercel Functions (API routes)
- **Database**: Supabase (Postgres)
- **AI**: LangChain + OpenAI
- **Deployment**: GitHub Pages + GitHub Actions
- **Security**: Snyk
